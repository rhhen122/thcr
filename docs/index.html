What is THCR? #

The Helix Config Repository is a Repository that contains Configs for Helix

Helix is a Vim-like Editor build in Rust. However it has a very cool feature called config.toml

Within this file you can set Keybinds, Themes and other quality-of-life improvements

The .toml makes Helix a more useable and better in general editor! But, some people dont want to spend ages making a config file. Eskopatramo this is where THCR comes in!

THCR is open-source and expandable! If you want to add to the repo then checkout the Github Repository and make a Fork, Make your changes and make a PR! Dont change the docs/index.html file or anything inside of the docs/ dir!

It may take a while for your PR to be accepted!

What if you want to be apart of THCR? Well you can apply by sending me a message via The contacts page
